SmartBrain

In the realm of academia and hands-on learning, the SmartBrain React App project stands out as a testament to my proficiency in building real-world applications. Focused on the intersection of React and Clarifai API integration, this project aimed to harness the power of facial recognition technology for practical applications. Leveraging React's component-based architecture, we designed an intuitive user interface that seamlessly interacts with external APIs to detect and recognize faces in uploaded images. The project not only showcases my technical acumen in frontend development but also highlights my ability to integrate complex functionalities, demonstrating a keen understanding of API interactions. This venture into facial recognition technology within the educational setting not only represents my dedication to mastering cutting-edge technologies but also underscores my passion for applying theoretical knowledge to real-world, hands-on projects.

Child-Help Kenya

Contributing to the greater good, the Child-Help Kenya website project holds a special place in my portfolio. It reflects my commitment to using technology for positive impact. The project not only showcases my technical skills but also demonstrates my dedication to leveraging technology as a force for positive change in the service of others.
